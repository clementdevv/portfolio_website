Projects

Handwritten Digit Classification with Artificial Neural Networks

This is a project based on Artificial Neural Networks using The Julia Programming Language to classify handwritten digits, using Julia's Flux.jl deep learning framework for model training. The MLDatasets,jl library also comes in handy for image recognition. The model is based on a simple neural network architecture, and it's training process is optimized using the RMSProp algorithm, with its performance being viewed on a learning curve plot, thanks to the Plots.jl library. The model is therefore able to perform highly accurate digit recognition.
